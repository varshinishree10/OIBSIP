# My Portfolio Website

This is my personal portfolio website created using **HTML, CSS, and Bootstrap 5**.  
It represents my profile, skills, projects, and contact information in a simple and responsive design.

---

## About the Portfolio

This portfolio highlights:
- My introduction as a web developer
- Projects I have worked on
- Key features of my work
- Testimonials
- Contact details and social media links

The website is fully responsive and designed with a modern UI.

---

## Sections Included

- **Navigation Bar** – Quick access to all sections  
- **Hero Section** – Introduction with profile image  
- **Features Section** – Clean code, responsive design, and performance  
- **Portfolio Section** – Showcases my projects  
- **Testimonials Section** – Feedback and reviews  
- **Contact Section** – Contact form  
- **Footer** – Contact details and social links
